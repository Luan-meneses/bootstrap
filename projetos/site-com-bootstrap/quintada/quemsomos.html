Quem Somos

Quitanda Online é uma loja virtual (e-commerce) fictícia criada como estudo de caso para o curso "Desenvolvimento de uma loja virtual com Bootstrap 5 e ASP.NET 5", disponibilizado gratuitamente pelo Prof. Ricardo Maroquio em seu canal do YouTube.

O código-fonte pode ser obtido via GitHub, conforme as orientações apresentadas ao longo dos vídeos do curso no YouTube. Este estudo de caso pode ser utilizado livremente como base para outras lojas virtuais, sem ônus para o usuário.

Bons estudos e boa programação!
